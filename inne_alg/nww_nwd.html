NWW i NWD

Co to są NWD i NWW?

NDW - Największy Wspólny Dzielnik dwóch liczb, na przykład:
NWD liczb 18 i 24 jest 4.

NWW - Najmniejsza Wspólna Wielokrotność dwóch liczb, na przykład:
NWW liczb 18 i 24 jest 72.

Algorytm NWD w Python metodą odejmowania

Ogólnie NWD dwóch liczb możemy obliczyć na 2 sposoby: metodą odejmowania i motodą modulo (reszta z dzielenia).

Metoda odejmowania:

a, b = map(int, input().split())
while a != b:
   if a > b:
     a = a - b
   if b > a:
     b = b - a
print(f"NWD wprowadzonych liczb = {a}")

Algorytm NWD. metoda modulo

Modulo - reszta z dzielenia. W Pythonie ma znak "%"

Metoda modulo:

a, b = map(int, input().split())
while b > 0:
   a, b = b, a % b
print(f"NWD wprowadzonych liczb = {a}")

Algorytm NWW

Jest bardziej prosty niż NWD, ale potrzebujemy NWD liczb do obliczenia ich NWW.

Trzeba zauważyć, że przed obliczeniem NWD wyliczamy iloczyn podanych liczb.

a, b = map(int, input().split())
iloczyn = a * b
while b > 0:
   a, b = b, a % b
nwd = a
nww = iloczyn / nwd
print(f"NWD wprowadzonych liczb = {nwd}")
print(f"NWW wprowadzonych liczb = {nww}")
