Karty Pracy 1

Zadanie 1

Napisz program, który obliczy sumę kwadratów dwóch wpisanych liczb.
Wejście: a, b
Wyjście: a2 + b2

a = int(input())
b = int(input())
print(a**2 + b**2)

Zadanie 2

Napisz program, który obliczy kwadrat sumy dwóch wpisanych liczb.
Wejście: a, b
Wyjście: (a + b)2

a = int(input())
b = int(input())
print((a + b)**2)

Zadanie 3

Napisz program, który obliczy sześcian różnicy dwóch wpisanych liczb.
Wejście: a, b
Wyjście: (a - b)3

a = int(input())
b = int(input())
print((a-b)**3)

Zadanie 4

Napisz program, który obliczy iloczyn trzech wpisanych liczb.
Wejście: a, b
Wyjście: a × b × c

a, b, c = map(int, input().split())
print(a*b*c)

Zadanie 5

Napisz program, który obliczy dwadzieścia procent podwojonej sumy dwóch wpisanych liczb.
Wejście: a, b, c
Wyjście: 2 × (a + b) / 5
Wyjście2: (a + b) × 0,4

a = int(input())
b = int(input())
print(round(2*(a+b)/5, 2))

Zadanie 6

Napisz program, który obliczy cenę netto wpisanej ceny (VAT 23%).
Wejście: brutto
Wyjście: brutto / 1,23

brutto = float(input())
print(brutto / 1.23)

Zadanie 7

Napisz program, który obliczy resztę z dzielenia dwóch wpisanych liczb.
Wejście: a, b
Wyjście: a % b (a modulo b)

a, b = int(input()), int(input())
print(a % b)
