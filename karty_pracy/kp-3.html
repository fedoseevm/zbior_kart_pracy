Zadanie 1

Napisz program, który wypisze ciąg sześcianów liczb naturalnych powiększonych o 3
Wejście: -
Wyjście: 3, 4, 11...

n = int(input("Podaj ilosc liczb: "))
for i in range(n):
  print(i**3 + 3, end=" ")

Zadanie 3

Napisz program, który wypisze wszyskie dzielniki podanej liczby n.
Wejście: n
Wyjście: dzielniki

n = int(input("Podaj liczbę n:\n"))
for i in range(1, n+1):
  if n%i == 0:
    print(i, end=" ")

Zadanie 5

(Hard) Bajtuś postanowił zrobić dla swoich kolegów pewną zagadkę. Zagadka polega na tym, że koledzy najpierw podają liczbę naturalną n. Następnie Bajtuś wybiera sobie liczbę x, taką, że 0 < x < n oraz wypisuje n - 1 różnych, mniejszych on n liczb pomijając przy tym swoje tajemnicze x. Napisz program, który pozwoli sprawdzić, jaką liczbę x wybrał sobie Bajtuś.
Przykłady:
    Wejście: 5
    1 2 4 5
    Wyjście: 3
    
    Wejście: 6
    6 5 3 2 1
    Wyjście: 4
    
    Wejście: 7
    4 6 1 3 7 2
    Wyjście: 5

n = int(input("Podaj ilość liczb: "))
s = n*(n+1)/2
for i in range(n-1):
  x = int(input())
  s = s - x
print(s)

Zadanie 6

(Hard) Napisz program według wyjścia poniżej (ciąg Fibonacciego). Zastosuj do tego zadania pętle for.
Wejście: -
Wyjście: 1, 2, 3, 5, 8, 13, 21, 34, ...

n = int(input())
a=1
b=2
print(a, b, end=" ")
for i in range(n-2):
  a,b=b, a+b
  print(b, end=" ")
